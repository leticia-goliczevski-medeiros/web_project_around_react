# Around the U.S. com React

Nome do projeto: Around the U.S. com React

Descrição do projeto: É uma página interativa, onde o usuário pode adicionar, remover ou curtir fotos. Ele também pode criar um perfil com informações básicas.

Tecnologias utilizadas: Media Query para a parte de responsividade, Grid Layout e Flexbox para o posicionameto de elementos. Toda a parte de estilos foi escrita em CSS puro. O React foi utilizado na renderização dos componentes e controle de estado.

Melhorias futuras: Validação dos formulários, adição de recurso de criação das postagens da página e alteraçôes das informações do usuário.
